### 📝 Question Entry Flow (مسار إدخال الأسئلة) — DETAILED USER FLOW

When a user clicks "إدخال الأسئلة", they go through this step-by-step flow. Make it feel smooth and guided — like a wizard/stepper UI with a progress bar at the top.

Step 1 — Course Info (معلومات المقرر):
Three fields on one page:
1. اسم الكلية — Dropdown list (e.g., كلية إدارة الأعمال، كلية الهندسة، كلية العلوم والدراسات الإنسانية، كلية هندسة وعلوم الحاسب، كلية التربية، كلية الصيدلة، كلية الطب، كلية التمريض، كلية طب الأسنان، كلية العلوم الطبية التطبيقية، أخرى)
2. القسم — Text input field (the student types their department name)
3. اسم المقرر — Text input field (the student types the course name)

After filling these, click "التالي" to go to Step 2.

Step 2 — Question Details (تفاصيل السؤال):

4. نص السؤال — Large text area for typing the question
5. رفع صورة (اختياري) — An optional image upload button for complex questions that are hard to type (supports jpg, jpeg, png, gif, webp). Show a preview of the uploaded image. Label it: "إذا لا يمكنك كتابة السؤال، ارفع صورة له"

6. الفصل (Chapter) — Dropdown list with options: Chapter 1 through Chapter 20, + "أخرى" (other). Next to it, an optional text field labeled "الموضوع (اختياري)" for the student to type the specific topic if they know it.

7. نوع السؤال — Dropdown list with these exact options:
   - اختيار من متعدد
   - صح وخطأ
   - أكمل الفراغ
   - أسئلة التوصيل
   - إجابة قصيرة
   - إجابة طويلة
   - مسائل رياضية / حسابية
   - صحّح الخطأ
   - نوع آخر

8. مستوى الصعوبة — Dropdown list:
   - سهل
   - متوسط
   - صعب

9. السنة — Dropdown list starting from 2023 up to 2026 (current year)

10. نوع الاختبار — Dropdown list:
    - ميد 1 (Midterm 1)
    - ميد 2 (Midterm 2)
    - فاينل (Final)

11. رابط مصدر أو شرح (اختياري) — Text input field. Label it: "أضف رابط لشرح السؤال أو الموضوع إن وجد (مقطع يوتيوب، ملخص، أي مصدر مجاني)". Show a small hint text below: "شارك معرفتك مع زملائك! 💡"

After filling all required fields, the user clicks "إرسال السؤال".

After Submission:
- Show a success message: "تم إرسال السؤال بنجاح وهو الآن قيد المراجعة ✅"
- Show a status badge: 🟡 قيد المراجعة
- Give the user two buttons:
  - "أضف سؤال آخر" — clears the question fields (Step 2) but keeps the course info (Step 1) so they don't have to re-enter college/department/course
  - "العودة للصفحة الرئيسية"
